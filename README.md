# Hibernate ORM Student Management Demo

Educational Java project demonstrating Hibernate ORM fundamentals with student entity management.

## Features
- **Student Entity Management**: CRUD operations for students
- **Hibernate Configuration**: XML and annotation-based setup
- **Database Operations**: ORM-based data persistence
- **Session Management**: Hibernate session handling
- **Transaction Management**: Proper commit/rollback

## Technology Stack
- Java
- Hibernate ORM
- Maven
- MySQL/SQL Database
- JPA Annotations

## Learning Objectives
- Understanding ORM concepts
- Hibernate configuration
- Entity mapping
- Database operations without SQL
- Session factory patterns

## Use Cases
- Hibernate learning project
- ORM demonstration
- Database abstraction example
- Educational reference
